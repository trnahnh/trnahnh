### What I'm shipping

**[Ferrox](https://github.com/trnahnh/ferrox)** - Order matching engine in Rust. 500ns P99 latency. 4.7M orders/sec. Zero hot-path allocations. Lock-free SPSC ring buffers. mmap WAL for crash recovery under 1.4ms.

**[Draft-Thinker](https://github.com/trnahnh/draft-thinker)** - Cost-aware LLM gateway in Go. Entropy-based routing, speculative execution via goroutines, semantic cache layer.

**[Inyeon](https://github.com/trnahnh/inyeon)** - Agentic AI git assistant. FastAPI + LangGraph + ChromaDB. 7-agent orchestration pipeline. 100ms median response. 220+ test cases at full coverage.

**[KatanaID](https://katanaid.com)** - AI branding toolkit in Go. 19+ concurrent API calls via goroutines. Sub-200ms responses. Gemini AI integration. Trust scoring + browser fingerprinting.

**[Caphne](https://caphne.co)** - Real-time study matching platform. 1,700+ req/min. Socket.IO + PostgreSQL + Redis. 400+ active users at FPT University.
